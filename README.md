# dva-hackernews

HackerNews clone built with [Dva](https://github.com/dvajs/dva), based on [vue-hackernews-2.0](https://github.com/vuejs/vue-hackernews-2.0).

## Run Locally

Install dependencies.

```bash
$ npm install
```

Start server.

```bash
$ npm start
```

Open app in browser.

```bash
$ open http://localhost:8989/
```
